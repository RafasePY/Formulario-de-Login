# Formulário de Login

Projeto desenvolvido utilizando **HTML5** e **CSS3**, com o objetivo de praticar a criação de interfaces modernas para autenticação de usuários.

## 📌 Funcionalidades

-   Campo de usuário
    
-   Campo de senha
    
-   Opção **Remember me**
    
-   Link de recuperação de senha
    
-   Botão de login estilizado
    
-   Ícones de redes sociais
    
-   Interface moderna e intuitiva
    

## 🛠️ Tecnologias Utilizadas

-   HTML5
    
-   CSS3
    
-   Feather Icons
    

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de desenvolvimento Front-End, incluindo:

-   Estruturação de páginas com HTML
    
-   Estilização com CSS
    
-   Criação de formulários
    
-   Posicionamento de elementos
    
-   Organização de componentes visuais
